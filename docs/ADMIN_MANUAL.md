# Admin Manual

## Core admin responsibilities
- Create and manage student, teacher, course, attendance, invoice, library, and notice records.
- Use the **Admin Panel** at `/admin/` for direct model-level access.
- Use the dashboard and reports page for quick operational summaries.

## Recommended workflow
1. Log in as admin.
2. Create teachers.
3. Create students.
4. Create courses and assign teachers.
5. Add attendance records.
6. Create invoices and mark payments.
7. Publish notices.

## Admin panel
The Flask-Admin panel provides data-level access to all main models:
- Users
- Student profiles
- Teacher profiles
- Courses
- Attendance
- Invoices
- Library books
- Notices
- Chat inquiries
