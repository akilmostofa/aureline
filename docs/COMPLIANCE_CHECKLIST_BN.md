# বাংলা কমপ্লায়েন্স চেকলিস্ট

- শক্তিশালী `SECRET_KEY` ব্যবহার করুন
- প্রোডাকশনে PostgreSQL বা MySQL ব্যবহার করুন
- HTTPS ছাড়া WhatsApp, Messenger, Zoom callback চালু করবেন না
- bKash/Nagad লাইভ করার আগে অফিসিয়াল মার্চেন্ট অনুমোদন সম্পন্ন করুন
- অ্যাডমিন পাসওয়ার্ড পরিবর্তন করুন
- রেগুলার ডাটাবেজ ব্যাকআপ চালু করুন
- ব্যবহারকারী ভূমিকা অনুযায়ী অ্যাক্সেস সীমাবদ্ধ রাখুন
